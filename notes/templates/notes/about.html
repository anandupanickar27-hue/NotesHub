About GoFi

GoFi is an AI-powered personal knowledge management platform that combines semantic search, Retrieval-Augmented Generation (RAG), and Google's Gemini AI to transform personal notes into an intelligent knowledge base. Instead of simply storing information, GoFi understands your notes, organizes them intelligently, and lets you retrieve knowledge through natural conversations.

📘 About GoFi

GoFi is an AI-powered personal knowledge management system that helps you capture, organize, search and interact with your notes using Retrieval-Augmented Generation (RAG).

✨ Features

  • AI-powered note generation
  • Manual note creation
  • Automatic summaries and tags
  • Smart category organization
  • Search, sorting and filters
  • Pin important notes
  • Chat with your own notes using AI
  • Private knowledge base for every user

⚙ Technology Stack

  • Django
  • Python
  • MySQL
  • LangChain
  • Google Gemini
  • ChromaDB
  • Bootstrap
  • HTML
  • CSS & JavaScript

🚀 How GoFi Works

  1. Create a note manually or with AI.
  2. AI organizes your content.
  3. Your notes are stored securely.
  4. Each note is converted into vector embeddings.
  5. GoFi retrieves the most relevant notes.
  6. Gemini generates answers using only your notes.

🔒 Privacy

Every user's notes are isolated. GoFi searches only your personal knowledge base when answering questions.
